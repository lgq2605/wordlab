📄 CSV Format Guide

How to create a CSV file manually and use it to practice.

Columns

ColumnRequired?DescriptionExample
rootRequiredThe base/root word (uppercased automatically)ABBREVIATE
wordOptionalA specific word formabbreviation
phoneticOptionalPronunciation in any format/əˌbriːviˈeɪʃən/
typeOptionalWord type: n, v, adj, adv, phr.vn
vietnamese_meaningOptionalVietnamese meaning shown during quizSự viết tắt

Example

root,word,phonetic,type,vietnamese_meaning
ABBREVIATE,abbreviate,/əˈbriːvieɪt/,v,Viết tắt
ABBREVIATE,abbreviation,/əˌbriːviˈeɪʃən/,n,Sự viết tắt
ACCESS,access,/ˈækses/,v,Truy cập
ACCESS,accessible,/əkˈsesɪbəl/,adj,Có thể truy cập

Word Types

n — Noun v — Verb adj — Adjective adv — Adverb phr.v — Phrasal Verb

How to Use Your File

Option A — Import into Database
Go to 🗂 Database → ⬆ Import CSV. Words merge into the main database with Weak Words tracking.
Option B — Session-only Practice
Go to 📝 Mode 1 or 🧩 Mode 2 → 📂 My Files. Nothing is saved to the database.

Tips

• Each root word can have as many rows as needed — one per word form.
• You can import multiple CSV files at once — they merge automatically.
• Column order doesn't matter — matched by header name.
• Extra columns are safely ignored.
